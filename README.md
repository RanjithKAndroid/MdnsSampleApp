# MdnsSampleApp

Android application demonstrating mDNS (NSD) service discovery on a local Wi-Fi network.

## Features
- Discover services using Android NSD (mDNS)
- Resolve device IP and service name
- Cache devices using Room database
- Coroutine-based background processing

## Tech Stack
- Kotlin
- XML
- Android NSD
- Room Database

## How to Run
- Clone the repo
- Open in Android Studio
- Run on a real Android device connected to Wi-Fi

## Notes
- mDNS works only on local network
- Emulator may not discover services
